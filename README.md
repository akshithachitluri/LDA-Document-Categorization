# Document Categorization using LDA

## Overview

This project implements Latent Dirichlet Allocation (LDA) for topic modeling and document categorization. The model analyzes text documents, extracts meaningful topics, and categorizes documents based on their dominant topics.

## Features

* Text preprocessing using NLTK and SpaCy
* Tokenization, stop-word removal, and lemmatization
* Topic modeling using Gensim's LDA algorithm
* Document categorization based on extracted topics
* Topic visualization and interpretation

## Technologies Used

* Python
* NLTK
* SpaCy
* Gensim
* Scikit-learn
* Jupyter Notebook

## Methodology

1. Data Collection
2. Text Preprocessing
3. Dictionary and Corpus Creation
4. LDA Model Training
5. Topic Extraction
6. Document Categorization
7. Topic Analysis and Visualization

## Results

The model successfully identified meaningful topics from text documents and categorized documents according to topic distributions. The project demonstrates practical applications of Natural Language Processing (NLP) and unsupervised machine learning techniques.
